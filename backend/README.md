# Backend

Esta carpeta contendrá el backend de IntegraTronic.

## Tecnología prevista

- Java.
- Spring Boot.
- API REST.
- Arquitectura por capas: Controller, Service y Repository.

## Alcance previsto

El backend dará soporte a los módulos definidos para el proyecto: usuarios y roles, productos, familias de productos, stock, movimientos de stock, ventas físicas, líneas de venta, clientes, pedidos online, líneas de pedido, pago online gestionado mediante sistema externo e informes básicos.

Todavía no se ha creado la estructura del proyecto ni se han fijado versiones definitivas.
