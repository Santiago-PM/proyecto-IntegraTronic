# Base de datos

Esta carpeta contendrá la documentación y futuros scripts relacionados con la base de datos de IntegraTronic.

## Tecnología prevista

- MySQL.

## Alcance previsto

La base de datos dará soporte a los módulos del proyecto: usuarios y roles, productos, familias de productos, stock, movimientos de stock, ventas físicas, líneas de venta, clientes, pedidos online, líneas de pedido, pago online gestionado mediante sistema externo e informes básicos.

Todavía no se ha creado el esquema SQL definitivo.
