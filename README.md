# IntegraTronic

IntegraTronic es un Trabajo de Fin de Grado desarrollado en equipo. El proyecto consiste en un ERP para una tienda física y online de productos informáticos.

## Tecnologías previstas

- Backend: Java + Spring Boot.
- Frontend: React.
- Base de datos: MySQL.
- Comunicación: API REST.
- Arquitectura backend: Controller, Service, Repository.

Las versiones definitivas de las tecnologías todavía no se han decidido.

## Módulos previstos

- Usuarios y roles.
- Productos.
- Familias de productos.
- Stock.
- Movimientos de stock.
- Ventas físicas.
- Líneas de venta.
- Clientes.
- Pedidos online.
- Líneas de pedido.
- Pago online gestionado mediante sistema externo.
- Informes y estadísticas básicas.

## Estructura inicial

- `backend/`: futuro proyecto backend.
- `frontend/`: futuro proyecto frontend.
- `database/`: documentación y scripts de base de datos.
- `docs/`: documentación del TFG.
